About This Server

This website is hosted on a secure Linux Virtual Machine running on Microsoft Azure. All updates are deployed automatically via a continuous delivery pipeline using GitHub Actions.

Every time a new commit is pushed to the main branch, the workflow triggers a secure SSH deployment process, updates the Apache web root, and publishes the latest version of the site.
