Slawinski Website

Deploy Automático via GitHub Actions → Azure VM

Este website está a correr numa máquina virtual Linux (Ubuntu) hospedada na Azure e atualizado automaticamente através de um pipeline de CI/CD com GitHub Actions.

Sempre que um novo commit é feito no branch main, o conteúdo é enviado automaticamente para este servidor.
